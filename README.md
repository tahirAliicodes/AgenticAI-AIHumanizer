## 🚀 What This Project Does

An **AI text humanizer** — takes AI-generated text and rewrites it to sound more natural and human-like. Supports text and Word document (.docx) input/output.

---

## ⚙️ Tech Stack

| Layer | Technology |
|---|---|
| **Language** | Python 3.11 |
| **LLM** | Ollama llama3.1 (local) |
| **API** | FastAPI |
| **File Support** | .txt, .docx |
| **Storage** | SQLite history database |

---

## ✅ Features

- ✍️ **Text humanizer** — rewrites AI text to sound natural
- 📄 **Document support** — accepts .txt and .docx files
- 💾 **History tracking** — SQLite database stores all humanizations
- 🌐 **Web UI** — clean HTML interface
- 📦 **Chunker** — handles large documents by splitting into chunks
- 🔧 **Assembler** — reassembles chunks after processing

## 🚀 Run Locally

```bash
git clone https://github.com/tahirAliicodes/AgenticAI-AIHumanizer.git
cd AgenticAI-AIHumanizer
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

Visit: `http://localhost:8000`
